# GAM Made Simple

A simplified graphical user interface (GUI) for Google Admin Console management using GAM (Google Apps Manager).

## What is GAM Made Simple?

GAM Made Simple is a desktop application that provides an easy-to-use interface for common Google Workspace administration tasks. Instead of remembering complex GAM command-line syntax, administrators can use this GUI to perform routine tasks with just a few clicks.

## Key Features

### User Management
- Get user information and details
- List users in organizational units (OUs)
- Force sign-out users from all devices
- Generate comprehensive user reports with group memberships, last login data, and OU information
- Multi-OU user management with easy selection interface

### Group Management
- List groups within organizational units
- View group members, owners, and managers
- Generate detailed group reports
- Export group data to CSV format

### Drive Management
- Transfer file ownership between users
- View drive usage and file permissions

### Calendar Management
- Search calendar events by query or date range
- View today's events for any user
- Delete specific calendar events
- List user calendars

### Delegate Management
- Add email delegates for users
- View existing delegates

### Custom Commands
- Execute any GAM command directly through the interface
- Save command output to files

## Why Use GAM Made Simple?

- **User-Friendly**: No need to remember complex GAM command syntax
- **Visual Interface**: Easy-to-navigate tabbed interface for different functions
- **Batch Operations**: Handle multiple users or groups at once
- **Error Prevention**: Input validation helps prevent common mistakes
- **Output Management**: Built-in output viewing and saving capabilities

## Requirements

- GAM (Google Apps Manager) must be installed and configured on your system
- Python with tkinter (typically included with Python installations)
- Appropriate Google Workspace administrator permissions
